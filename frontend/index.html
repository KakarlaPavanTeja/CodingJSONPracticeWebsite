How to Use

01

Select Type

Choose "Standard" or "Node-Based" from the sidebar. This determines the code structure.

02

Choose Action

Select "Create New" for fresh content, or "Update" to modify existing JSONs while keeping IDs.

03

Upload Files

Upload your .lua config and test cases .json. For updates, include the old JSON.

04

Generate

Click Generate. The tool handles parsing, validating, and bundling into the final JSON.

Content Generation

  • Follow-up Questions: Automatically generates interview follow-up questions based on the provided solution.
  • Granular Regeneration: Use the "Regenerate" buttons to refresh specific sections (Hints, Real-life Examples, or Follow-ups) without re-running the entire prompt.

  • Dynamic Currency: Real-time USD to INR conversion for accurate cost tracking using live exchange rates.
