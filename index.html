Reference & Guides

How to Use

  1. Choose Question Type: On the home page, select "Standard" or "Node-Based".
  2. Choose Action: Select "Create New" for a new question or "Update Existing" to modify one.
  3. Prepare Your Files: Use the outline templates linked below to prepare your `.lua` and `.json` files.
  4. Upload Files: Upload all required files. The "Update" action requires the existing JSON file as well.
  5. Generate & Download: Click the button to receive your final `coding_questions.json` file.

Tool Features

  • Create & Update: Supports both generating new question JSONs and updating existing ones while preserving key IDs.
  • Dual Question Type Support: Works for both Standard and Node-Based (Linked List, Binary Tree) questions.
  • Automatic Weightage Calculation: Test case weights are calculated for you based on question difficulty.
  • Comprehensive Parsing: Handles all metadata, hints, follow-up questions, and debug code.

Hint Section Format

The tool automatically generates hint titles (e.g., "Hint 1", "Hint 2") and sets the unlock time to 5 seconds. You only need to provide the description content for each hint in your .lua file as follows:

----------HINTS_START----------

----------HINTS_START_1----------
Description for the first hint. This can be multi-line and supports Markdown.
----------HINTS_END_1----------

----------HINTS_START_2----------
Description for the second hint.
----------HINTS_END_2----------

----------HINTS_END----------
